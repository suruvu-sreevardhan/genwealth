Place your logo files in this folder with these exact names:

1) icon_logo_1024.png
   - Use icon logo only
   - 1024x1024 master
   - Keep safe padding around symbol
   - Monochrome (black/white) recommended

2) full_logo.png
   - Use full logo (icon + finlit text)
   - Transparent background preferred

Optional manual exports if needed:
- icon_logo_512.png  (512x512)
- icon_logo_192.png  (192x192)
- icon_logo_48.png   (48x48)

After adding files, run:
flutter pub get
flutter pub run flutter_launcher_icons

Then rebuild the app.
